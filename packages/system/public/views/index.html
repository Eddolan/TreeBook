Treebook

TreeBook is a social network for trees. We have a profile page for each registered tree in San Francisco so come make some new sprouting friends. Grow your social branches by connecting with some perrenial plants. The code that runs this site can be found on github.

Explore the Forest

Each tree has a profile page like the beautiful Alicia Higgins. Each profile has basic information and map showing its precice location. The profile pictures are of each tree species, but if you want to see the exact tree you can use the street view functionality of the embedded map.

Browse all trees

About the Project

The project was created in a week long sprint by 4 students at Hack Reactor. We use the MEAN stack and also used SQL to store the tree data.
